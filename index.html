PrimerMapper

GUI for batch primer design with graphical output for PCR and SNP detection

Welcome to GitHub Pages.

PrimerMapper PrimerMapper Provides the following: 1. a GUI to facilitate primer design and visualization 2. Traditional primer design as well as SNP and allele specific primer design 3. Visualization of primer distribution across each sequence and entire input file 4. Returns primers specific to the entire input file using specificty and mis-match options 5. Remote sequence access from GenBank and dbSNP 6. Primer BLAST facility against multiple NCBI databases 7. Generates primer dimer scores for all primers generated to facilitate multiplex PCR expts using GitHub Flavored Markdown, select a template crafted by a designer, and publish.
